# รายการ ราคา (฿)
1 เชื่อมต่อระบบกับ Tikfinity ให้ต่ออายุ Pro ให้ลูกค้าอัตโนมัติ ทันทีหลังลูกค้าชำระเงิน 1,000
2 ระบบรับชำระเงินผ่าน บัตรเครดิต และ พร้อมเพย์ (ผ่าน Stripe ผู้ให้บริการรับชำระเงินระดับโลก) 1,000
3 หน้าเว็บสำหรับลูกค้า — กรอกอีเมล → เลือกแพ็กเกจ → ยืนยัน → ชำระเงิน (ขั้นตอนเดียวจบ) 1,200
4 หน้าแอดมินหลัก — กราฟยอดขาย 14 วันล่าสุด สถิติแพ็กเกจขายดี และเครดิตคงเหลือใน Tikfinity 1,000
5 หน้าตั้งราคาแพ็กเกจ — แอดมินตั้งราคา/ชื่อ/เปิด-ปิดได้เอง + ปุ่มดึงราคา USD จากอัตราแลกเปลี่ยนสดอัตโนมัติ 600
6 หน้าจัดการคำสั่งซื้อ — ดูรายการ ดูรายละเอียด + ปุ่มต่ออายุให้ลูกค้าด้วยมือ (กรณีระบบขัดข้อง) 600
7 ระบบสลับภาษา ไทย / อังกฤษ รองรับลูกค้าต่างชาติ 400
8 ออกแบบหน้าเว็บให้สวยงาม โหมดสว่าง/มืด เข้าธีม Judy Studio + ระบบทดสอบความถูกต้องของโค้ด 500
ราคารวมทั้งสิ้น / Total 6,300 ฿
